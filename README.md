# DynEmit
A NuGet for simply generate ASM through EMIT in .NET with optimizing performance.  
使用IL动态生成字节码的NuGet包，可以大幅优化程序部分场景性能和创造更多可能。

# 特点  
- 低成本学习  
- API友好，对底层封装完善，几乎零接触OpCodes等低代码  
- 几乎无损耗操作，通过在编译期解决个性化需求来优化程序性能，且大幅优化反射性能  
- 更为友好的编写逻辑，及时抛出报错。
